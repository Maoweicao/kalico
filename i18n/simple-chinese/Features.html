    功能¶

    Kalico 有多个引人注目的功能：

    • 高精度步进电机运动。Kalico 利用应用处理器（例如低成本 Raspberry Pi）来计算打印机运动。应用处理器确定何时对每个步进电机进行步进，压缩这些事件，将其传输到微控制器，然后微控制器在请求的时间执行每个事件。每个步进事件都以 25 微秒或更好的精度进行调度。该软件不使用运动学估计（例如 Bresenham 算法） - 相反，它根据加速度的物理和机器运动学的物理计算精确的步进时间。更精确的步进电机运动提供更安静和更稳定的打印机操作。
    • 同类最佳性能。Kalico 能够在新旧微控制器上实现高步进速率。即使是旧的 8 位微控制器也可以获得 175K 步/秒以上的速率。在较新的微控制器上，每秒可以达到数百万步。更高的步进速率启用更高的打印速度。即使在高速下，步进事件时序也保持精确，这改进了整体稳定性。
    • Kalico 支持具有多个微控制器的打印机。例如，一个微控制器可用于控制挤出机，另一个控制打印机的加热器，而第三个控制打印机的其余部分。Kalico 主机软件实现时钟同步以解释微控制器之间的时钟漂移。不需要特殊代码来启用多个微控制器 - 它只需要在配置文件中需要几行额外的内容。
    • 通过简单配置文件进行配置。无需重新刷新微控制器来改变设置。Kalico 的所有配置都存储在标准配置文件中，可以轻松编辑。这使得设置和维护硬件变得更容易。
    • Kalico 支持"光滑压力提前" - 一种机制来解释挤出机内的压力的影响。这减少了挤出机"渗出"并改进了打印角的质量。Kalico 的实现不会引入瞬间挤出机速度变化，这改进了整体稳定性和健壮性。
    • Kalico 支持"输入整形"以减少振动对打印质量的影响。这可以减少或消除打印中的"振铃"（也称为"幽灵"、"回声"或"涟漪"）。它也可能允许人获得更快的打印速度，同时仍然保持高打印质量。
    • Kalico 使用"迭代求解器"从简单的运动学方程计算精确的步进时间。这使得将 Kalico 移植到新型机器人变得更容易，并且即使具有复杂的运动学也保持时序精确（不需要"线段"）。
    • Kalico 是硬件不可知的。无论低级电子硬件如何，都应该获得相同的精确时序。Kalico 微控制器代码旨在忠实地遵循 Kalico 主机软件提供的时间表（或突出提醒用户是否无法）。这使得更容易使用可用的硬件、升级到新硬件并对硬件有信心。
    • 便携式代码。Kalico 适用于 ARM、AVR、PRU 和其他微控制器。现有的"reprap"风格打印机可以在没有硬件修改的情况下运行 Kalico - 只需添加 Raspberry Pi。Kalico 的内部代码布局也使得支持其他微控制器体系结构变得更容易。
    • 更简单的代码。Kalico 对大多数代码使用非常高级的语言（Python）。运动学算法、G 代码解析、加热和热敏电阻算法等都用 Python 编写。这使得开发新功能变得更容易。
    • 自定义可编程宏。新的 G-Code 命令可以在打印机配置文件中定义（不需要代码更改）。这些命令是可编程的 - 允许它们根据打印机的状态产生不同的操作。
    • 内置 API 服务器。除了标准 G-Code 接口，Kalico 还支持丰富的基于 JSON 的应用接口。这使程序员能够构建对打印机进行详细控制的外部应用。

    其他功能¶

    Kalico 支持许多标准的 3D 打印机功能：

    • 多个网络界面可用。适用于 Mainsail、Fluidd、OctoPrint 等。这允许使用常规网络浏览器控制打印机。运行 Kalico 的同一 Raspberry Pi 也可以运行网络界面。
    • 标准 G-Code 支持。支持由典型"切片器"（SuperSlicer、Cura、PrusaSlicer 等）生成的常见 G 代码命令。
    • 支持多个挤出机。具有共享加热器和独立车体上的挤出机（IDEX）的挤出机也得到支持。
    • 支持笛卡尔、三角洲、corexy、corexz、混合 corexy、混合 corexz、deltesian、旋转三角洲、极坐标和绞盘风格打印机。
    • 自动床平整支持。Kalico 可配置为基本床倾斜检测或完整网格床平整。如果床使用多个 Z 步进器，Kalico 也可以通过独立操纵 Z 步进器来调平。大多数 Z 高度探针都得到支持，包括 BL-Touch 探针和伺服激活探针。探针可以针对轴扭转补偿进行校准。
    • 自动三角洲校准支持。校准工具可以执行基本高度校准以及增强的 X 和 Y 尺寸校准。校准可以使用 Z 高度探针或通过手动探测进行。
    • 运行时"排除对象"支持。配置后，此模块可能便于取消多部分打印中的仅一个对象。
    • 支持通用温度传感器（例如通用热敏电阻、AD595、AD597、AD849x、PT100、PT1000、MAX6675、MAX31855、MAX31856、MAX31865、BME280、HTU21D、DS18B20、AHT10 和 LM75）。也可以配置自定义热敏电阻和自定义模拟温度传感器。可以监视内部微控制器温度传感器和 Raspberry Pi 的内部温度传感器。
    • 默认启用基本热加热器保护。
    • 支持标准风扇、喷嘴风扇和温度控制风扇。打印机闲置时无需保持风扇运行。可以在具有转速表的风扇上监视风扇速度。
    • 支持 TMC2130、TMC2208/TMC2224、TMC2209、TMC2240、TMC2660、TMC5160 和 TMC2160 步进电机驱动器的运行时配置。还支持通过 AD5206、DAC084S085、MCP4451、MCP4728、MCP4018 和 PWM 引脚传统步进驱动器的电流控制。
    • 支持直接连接到打印机的通用 LCD 显示器。默认菜单也可用。显示和菜单的内容可以通过配置文件完全自定义。
    • 恒定加速和"前瞻"支持。所有打印机移动将从静止状态逐渐加速到巡航速度，然后减速回到静止状态。传入的 G-Code 移动命令流被排队并分析 - 类似方向上的移动之间的加速将被优化以减少打印停滞并改进整体打印时间。
    • Kalico 实现了一种"步进器相位限位"算法，可以改进典型限位开关的精度。适当调整后，它可以改进打印第一层床附着力。
    • 支持灯丝存在传感器、灯丝运动传感器和灯丝宽度传感器。
    • 支持使用 adxl345、mpu9250、mpu6050 和 lis2dw12 加速度计测量和记录加速度。
    • 支持限制短"之字形"移动的最高速度以减少打印机振动和噪音。有关更多信息，请参阅 运动学 文档。
    • 示例配置文件可用于许多常见的打印机。检查 config 目录 以获取列表。

    要开始使用 Kalico，请阅读 安装 指南。
