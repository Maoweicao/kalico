# 自动 Z 偏移校准

本文档提供有关自动校准喷嘴 Z 偏移的信息。启用此功能后，无需进行手动 Z 偏移或第一层校准。它始终计算正确的偏移，与当前温度、使用的喷嘴、使用的打印床或柔性板无关。

# 为什么需要这个

- Voron V1 或 V2 打印机中使用的 Z 端点是一个聪明的，因为喷嘴点击固定在打印床上的开关。这允许在不改变偏移量（在开关和床之间）的情况下交换喷嘴：
  ![endstop offset](img/z_calibrate-endstop.png)
- 或者，通过使用基于表面的探针（如磁力探针）作为 Z 端点。这允许在不改变偏移的情况下交换柔性板：
  ![probe offset](img/z_calibrate-probe.png)
  感应式探针不会工作，因为它不直接探测床的表面！
- 但是，是否有可能同时获得两者？

这是可能的，这就是此扩展所做的！

# 要求

但是，使用它有一些要求：

- 喷嘴尖端在开关上驱动的 Z 端点（如库存 Voron V1/V2 端点）。它将无法与配置为端点的探针虚拟引脚一起工作！
- 打印头上的（磁性）基于开关的探针
- Z 端点和磁力探针都配置正确，回位和任何种类的床平铺都工作正常。
- 需要磁力探针的连接和分离宏来进行此配置。

# 它所做的

1. 使用 Z 端点进行所有轴的正常回位（这不是此插件的一部分）。之后，在 Z 中有一个定义的零点。从现在开始，一切都与这个点相关。因此，新的回位会改变一切，因为回位不是那么精确的。
2. 通过在 Z 端点上探测喷嘴尖端来确定喷嘴的高度（类似于第 1 步中的回位。但这可能会产生稍微不同的值）：
   ![nozzle position](img/z_calibrate-nozzle.png)
3. 通过在 z 端点上探测开关的主体来确定磁力探针的高度：
   ![switch position](img/z_calibrate-switch.png)
4. 计算喷嘴尖端和磁力探针触发点之间的偏移：

   `喷嘴开关偏移 = 磁力探针高度 - 喷嘴高度 + 开关偏移`

   ![switch offset](img/z_calibrate-offset.png)

   磁力探针的触发点无法直接探测。这就是为什么要在端点上点击开关的身体而不是触发凸起。这就是为什么使用小开关偏移来反映在触发时凸起和身体之间的偏移。
   此偏移是固定的。
5. 通过用磁力探针在床上探测一个点来确定打印表面的高度（最好是中心或"bed_mesh:relative_reference_index"的配置/使用的网格）。
6. 现在，最终偏移计算如下：

   `探针偏移 = 探测的高度 - 计算的喷嘴开关偏移`

7. 最后，通过使用 `SET_GCODE_OFFSET` 命令应用计算的偏移（之前的偏移被重置！）。

## 干扰

温度或湿度变化不是大问题，因为开关不受太大影响，所有值都在小时间周期内探测，只有相互之间的关系被使用。在第 2 步中喷嘴高度可以稍后确定，甚至与回位比较相比，打印机腔室中高出许多摄氏度。这就是为什么喷嘴再次被探测，可以稍微改变到第一个回位位置。

## 示例输出

具有所有确定位置的校准输出如下所示（偏移是应用为 GCode 偏移的偏移）：

```
Z-CALIBRATION: ENDSTOP=-0.300 NOZZLE=-0.300 SWITCH=6.208 PROBE=7.013 --> OFFSET=-0.170
```

端点值是回位的 Z 位置，始终为零或配置的"stepper_z:position_endstop"设置 - 在这种情况下，它甚至与探测的喷嘴高度相同。

# 配置

要激活扩展，打印机配置中需要 `[z_calibration]` 部分。配置属性在 [这里](Config_Reference.md#z_calibration) 详细描述。

## 开关偏移

"z_calibration:switch_offset" 是已提到的从开关主体（即探测位置）到实际触发点的偏移。可以从数据表（如欧姆龙开关（D2F-5：0.5mm 和 SSG-5H：0.7mm））中获取此值的起点。根据您对第一层想要的脆弱性（对于 D2F-5 大约 0.45），从略少于此开始是好的。因此，使用较小的偏移值，喷嘴离床更远！该值不能为负。

例如，D2F-5 的数据表：

![endstop offset](img/z_calibrate-d2f.png)

以及偏移基数的计算：

```
偏移基数 = OP（操作位置）- 开关主体高度
     0.5 毫米 = 5.5 毫米 - 5 毫米
```

## 连接和分离探针

磁力探针的连接和分离可以通过为 `CALIBRATE_Z` 命令创建宏并用适当的命令围绕它来完成：

```
[gcode_macro CALIBRATE_Z]
description: 自动校准喷嘴对打印表面的偏移并对接/分离 MagProbe
rename_existing: CALIBRATE_Z_BASE
gcode:
  ATTACH_PROBE     # 用您特定的连接宏名称替换
  CALIBRATE_Z_BASE
  DETACH_PROBE     # 用您特定的分离宏名称替换
```

也可以使用 `start_gcode` 和 `end_gcode` 属性来改为调用连接和分离命令：

```
[z_calibration]
...
start_gcode: ATTACH_PROBE  # 用您特定的连接宏名称替换
end_gcode: DETACH_PROBE    # 用您特定的分离宏名称替换
```

如果存在空间限制，无法在探针连接的情况下在端点上探测喷嘴，可以使用 `before_switch_gcode` 属性来改为连接探针。然后，探针在探针在端点上被探测之前不会连接：

```
[z_calibration]
...
before_switch_gcode: ATTACH_PROBE  # 用您特定的连接宏名称替换
end_gcode: DETACH_PROBE            # 用您特定的分离宏名称替换
```

## 床网格

如果使用床网格，打印床上探测的坐标必须恰好是网格的相对参考索引点，因为这是网格的零点！但是，可以完全省略这些属性，相对参考索引点的网格将被自动获取（为此，需要"bed_mesh:relative_reference_index"设置，目前不支持圆形床/网格）！

# 如何测试

不要太担心计算偏移的绝对值。这些可能差异很大。只有喷嘴到床的实际位置才算。要测试这一点，可以首先通过 `GET_POSITION` 查询校准结果：

```
> CALIBRATE_Z
> Z-CALIBRATION: ENDSTOP=-0.300 NOZZLE=-0.267 SWITCH=2.370 PROBE=3.093 --> OFFSET=-0.010000
> GET_POSITION
> mcu: stepper_x:17085 stepper_y:15625 stepper_z:-51454 stepper_z1:-51454 stepper_z2:-51454 stepper_z3:-51454
> stepper: stepper_x:552.500000 stepper_y:-47.500000 stepper_z:10.022500 stepper_z1:10.022500 stepper_z2:10.022500 stepper_z3:10.022500
> kinematic: X:252.500000 Y:300.000000 Z:10.022500
> toolhead: X:252.500000 Y:300.000000 Z:10.021472 E:0.000000
> gcode: X:252.500000 Y:300.000000 Z:9.990000 E:0.000000
> gcode base: X:0.000000 Y:0.000000 Z:-0.010000 E:0.000000
> gcode homing: X:0.000000 Y:0.000000 Z:-0.010000
```

这里，"gcode base" 中的 Z 位置反映了校准的 Z 偏移。

然后，可以通过在多个步骤中缓慢将喷嘴向下移动到零来测试偏移。通过使用 GCode 这样做是好的，因为偏移作为 GCode 偏移应用。例如像这样：

```
> G90
> G0 Z5
> G0 Z3
> G0 Z1
> G0 Z0.5
> G0 Z0.3
> G0 Z0.1
```

在每一步后检查与打印表面的距离。如果有小的差异（应该小于开关数据表中的偏移基数），那么通过该值调整"z_calibration:switch_offset"。
减少 "switch_offset" 会将喷嘴移离床更远。

最后，如果您已双重检查校准的偏移是正确的，您可以继续通过实际打印第一层测试来微调 "z_calibration:switch_offset"。这只需要做一次！

# 如何使用

## 命令 CALIBRATE_Z

通过使用 `CALIBRATE_Z` 命令启动校准。没有更多参数。运行此命令需要干净的喷嘴。

何时进行此校准无关紧要（以及在什么温度下）。但是，重要的是在打印机热的时候在启动打印前调用它。因此，将 `CALIBRATE_Z` 命令添加到 `PRINT_START` 宏（从切片软件的启动 GCode 调用）是一个好主意。此宏的序列可以如下所示：

1. 回位所有轴
2. 加热床和喷嘴（以及腔室）
3. 获取探针，如果需要进行任何床平铺（如 QGL、Z-Tilt），停放探针
4. 清除和清洁喷嘴
5. 获取探针，CALIBRATE_Z，停放探针
6. （如果需要调整 Z 偏移）
7. 打印介绍线（如果使用）
8. 开始打印...

**:exclamation: 并删除这里的任何旧 Z 偏移调整
(如 `SET_GCODE_OFFSET`)**

对于纹理打印表面，可能需要更接近床。
要从切片软件的启动 GCode 调整偏移，可以将以下命令添加到 `PRINT_START` 宏中**后**调用 Z 校准：

```
    # 如果需要，调整 G-Code Z 偏移
    SET_GCODE_OFFSET Z_ADJUST={params.Z_ADJUST|default(0.0)|float} MOVE=1
```

然后，可以在切片软件的 `PRINT_START` 命令中添加 `Z_ADJUST=0.0`。这**不**将偏移重置为此值，而是通过给定的金额调整它！

>**注意：** 运行 Z 校准后不要再次回位 Z，否则需要再次执行！

## 命令 PROBE_Z_ACCURACY

还有一个 `PROBE_Z_ACCURACY` 命令来测试 Z 端点的准确性（如探针的 `PROBE_ACCURACY` 命令）：

```
PROBE_Z_ACCURACY [PROBE_SPEED=<mm/s>] [LIFT_SPEED=<mm/s>] [SAMPLES=<count>] [SAMPLE_RETRACT_DIST=<mm>]
```

它通过获取配置的喷嘴在端点上的位置来计算多个探针样本上的最大值、最小值、平均值、中位数和标准差。可选参数默认为 z_calibration 配置部分中的等效设置。

## 渗出缓解

任何喷嘴探针端点的渗出可能导致不准确，因为丝材将继续在多个探针的空间上泄漏或变形。强烈建议采取一些措施防止此插件的喷嘴探针部分之前的渗出积累。

缓慢的长缩回，多达 15mm 在打印结束时，可以减少渗出积累的可能性。如果这样做，请考虑在您的打印启动序列中的最后一个命令中添加可比较的挤出，将塑料带回尖端。（大于 5mm 的缩回已与许多热端的堵塞有关，特别是 Rapido。这可能最好被认为是最后的手段，具体取决于精确的硬件和丝材。）

在刮洗 - 使用清洁桶 - 大约一分钟前加热喷嘴将允许所有剩余的塑料时间从喷嘴排出并被简单的擦拭清理。如果使用清洁和刮洗桶，此阶段不要排出丝材。

需要更强激活力的端点开关，例如性别螺栓与弹簧，或不粘 z，可以帮助压破任何剩余渗出并改善一致性。

探针可以在热端温度低于打印的完全温度时完成。如果您在 250 处打印，您可以将喷嘴预热至 180，并在完全温度完成前运行此脚本。这可能取决于使用的温度而产生不同的效果。

还应考虑在喷嘴擦拭之前拾取您的探针，以允许此脚本在清洁后立即探测喷嘴。
