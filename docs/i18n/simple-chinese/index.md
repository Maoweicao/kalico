# 欢迎来到 Kalico 文档！

![](logo/kalico-big.png){ .nobg .center-image .main-logo }

Kalico 是 Klipper 3D 打印机固件社区维护的分支。
它结合了通用计算机的强大功能与一个或多个
微控制器。有关为什么应该使用 Kalico 的更多信息，请参阅 [功能](Features.md) 文档。

要开始使用 Kalico，请开始 [安装](Installation.md) 它。如果您考虑从 Klipper 迁移到 Kalico，
请阅读我们的专用 [迁移指南](Migrating_from_Klipper.md)。

Kalico 是自由软件。阅读 [文档](Overview.md) 或
在 [GitHub 上查看 Kalico 代码](https://github.com/KalicoCrew/kalico)。

我们依赖于 [赞助商](Sponsors.md) 的慷慨支持。
