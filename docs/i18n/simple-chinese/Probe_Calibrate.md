# 探针校准

本文档描述了在Kalico中校准"自动z探针"的X、Y和Z偏移的方法。这对具有`[probe]`或`[bltouch]`配置部分的用户很有用。

## 校准探针X和Y偏移

要校准X和Y偏移，请导航到OctoPrint"控制"选项卡，归位打印机，然后使用OctoPrint摇杆按钮将头部移至床面中心附近的位置。

在探针下方的床面上放置一块蓝色画家胶带（或类似的东西）。导航到OctoPrint"终端"选项卡并发出PROBE命令：
```
PROBE
```
在探针正下方的胶带上放置一个标记（或使用类似的方法记下床面上的位置）。

发出`GET_POSITION`命令并记录该命令报告的工具头XY位置。例如，如果看到：
```
Recv: // toolhead: X:46.500000 Y:27.000000 Z:15.000000 E:0.000000
```
那么将记录探针X位置为46.5，探针Y位置为27。

记录探针位置后，发出一系列G1命令，直到喷嘴直接位于床上的标记上方。例如，可能会发出：
```
G1 F300 X57 Y30 Z15
```
将喷嘴移动到X位置57和Y位置30。找到标记正上方的位置后，使用`GET_POSITION`命令报告该位置。这是喷嘴位置。

然后x_offset为`nozzle_x_position - probe_x_position`，y_offset类似为`nozzle_y_position - probe_y_position`。使用给定的值更新printer.cfg文件，从床上删除胶带/标记，然后发出`RESTART`命令，使新值生效。

## 校准探针Z偏移

提供准确的探针z_offset对于获得高质量打印至关重要。z_offset是探针触发时喷嘴和床面之间的距离。Kalico`PROBE_CALIBRATE`工具可用于获取此值——它将运行自动探针来测量探针的Z触发位置，然后启动手动探针来获取喷嘴Z高度。然后从这些测量计算探针z_offset。

首先归位打印机，然后将头部移至床面中心附近的位置。导航到OctoPrint终端选项卡并运行`PROBE_CALIBRATE`命令启动该工具。

此工具将执行自动探针，然后抬起头部，将喷嘴移至探针点的位置上方，并启动手动探针工具。如果喷嘴未移动到自动探针点上方的位置，则`ABORT`手动探针工具并执行上面描述的XY探针偏移校准。

手动探针工具启动后，按照["纸张测试"](Bed_Level.md#the-paper-test)中描述的步骤确定给定位置处喷嘴和床面之间的实际距离。完成这些步骤后，可以`ACCEPT`位置并使用以下命令将结果保存到配置文件：
```
SAVE_CONFIG
```

注意，如果对打印机的运动系统、热端位置或探针位置进行了改动，将使PROBE_CALIBRATE的结果无效。

如果探针具有X或Y偏移，并且改变了床面倾斜（例如，通过调整床螺钉、运行DELTA_CALIBRATE、运行Z_TILT_ADJUST、运行QUAD_GANTRY_LEVEL或类似操作），则将使PROBE_CALIBRATE的结果无效。进行上述任何调整后，需要再次运行PROBE_CALIBRATE。

如果PROBE_CALIBRATE的结果无效，那么使用探针获得的任何以前的[床面网格](Bed_Mesh.md)结果也将无效——重新校准探针后需要重新运行BED_MESH_CALIBRATE。

## 可重复性检查

校准探针X、Y和Z偏移后，最好验证探针提供可重复的结果。首先归位打印机，然后将头部移至床面中心附近的位置。导航到OctoPrint终端选项卡并运行`PROBE_ACCURACY`命令。

此命令将运行探针十次并产生类似于以下的输出：
```
Recv: // probe accuracy: at X:0.000 Y:0.000 Z:10.000
Recv: // and read 10 times with speed of 5 mm/s
Recv: // probe at -0.003,0.005 is z=2.506948
Recv: // probe at -0.003,0.005 is z=2.519448
Recv: // probe at -0.003,0.005 is z=2.519448
Recv: // probe at -0.003,0.005 is z=2.506948
Recv: // probe at -0.003,0.005 is z=2.519448
Recv: // probe at -0.003,0.005 is z=2.519448
Recv: // probe at -0.003,0.005 is z=2.506948
Recv: // probe at -0.003,0.005 is z=2.506948
Recv: // probe at -0.003,0.005 is z=2.519448
Recv: // probe at -0.003,0.005 is z=2.506948
Recv: // probe accuracy results: maximum 2.519448, minimum 2.506948, range 0.012500, average 2.513198, median 2.513198, standard deviation 0.006250
```

理想情况下，工具将报告相同的最大值和最小值。（即，理想情况下探针在所有十次探针上获得相同的结果。）但是，最小值和最大值相差一个Z"步进距离"或最多5微米（0.005毫米）是正常的。"步进距离"为`rotation_distance/(full_steps_per_rotation*microsteps)`。最小值和最大值之间的距离称为范围。所以，在上面的例子中，由于打印机使用Z步进距离0.0125，0.012500的范围将被认为是正常的。

如果测试结果显示范围值大于25微米（0.025毫米），则探针对于典型的床面调平程序精度不足。可能可以通过调整探针速度和/或探针启动高度来改进探针的可重复性。`PROBE_ACCURACY`命令允许运行具有不同参数的测试以查看其影响——有关进一步的详细信息，请参阅[G代码文档](G-Codes.md#probe_accuracy)。如果探针通常获得可重复的结果但有偶发的离群值，那么可能可以通过在每次探针上使用多个样本来说明——在[配置参考](Config_Reference.md#probe)中阅读探针`samples`配置参数的描述以获取更多详细信息。

如果需要新的探针速度、样本计数或其他设置，则更新printer.cfg文件并发出`RESTART`命令。如果是这样，最好再次[校准z_offset](#calibrating-probe-z-offset)。如果无法获得可重复的结果，则不要对床面调平使用探针。Kalico有几个可以改用的手动探针工具——有关进一步的详细信息，请参阅[床面调平文档](Bed_Level.md)。

## 位置偏差检查

某些探针在某些工具头位置可能存在系统偏差，这会破坏探针结果。例如，如果沿Y轴移动时探针支架轻微倾斜，则可能导致探针在不同的Y位置报告有偏差的结果。

这是增量打印机上的常见问题，但它可能发生在所有打印机上。

通过使用`PROBE_CALIBRATE`命令在各种X和Y位置测量探针z_offset来检查位置偏差。理想情况下，探针z_offset在打印机的每个位置都是常数值。

对于增量打印机，尝试在A塔附近的位置、B塔附近的位置和C塔附近的位置测量z_offset。对于笛卡尔、CoreXY和类似打印机，尝试在床面的四个角附近的位置测量z_offset。

在开始此测试之前，首先按照本文档开头所述校准探针X、Y和Z偏移。然后归位打印机并导航到第一个XY位置。按照[校准探针Z偏移](#calibrating-probe-z-offset)中的步骤运行`PROBE_CALIBRATE`命令、`TESTZ`命令和`ACCEPT`命令，但不要运行`SAVE_CONFIG`。记录报告的z_offset。然后导航到其他XY位置，重复这些`PROBE_CALIBRATE`步骤，并记录报告的z_offset。

如果报告的最小z_offset和最大z_offset之间的差异大于25微米（0.025毫米），则探针不适合用于典型的床面调平程序。参阅[床面调平文档](Bed_Level.md)了解手动探针替代品。

## 温度偏差

许多探针在不同温度下探针时具有系统偏差。例如，当探针处于更高温度时，探针可能会在更低的高度始终触发。

建议以恒定的温度运行床面调平工具来说明此偏差。例如，要么在打印机处于室温时总是运行工具，要么在打印机达到恒定打印温度后总是运行工具。在任何一种情况下，最好在达到所需温度后等待几分钟，以便打印机装置始终处于所需的温度。

要检查温度偏差，从打印机处于室温开始，然后归位打印机，将头部移至床面中心附近的位置，并运行`PROBE_ACCURACY`命令。记录结果。然后，不进行归位或禁用步进电机，将打印机喷嘴和床加热到打印温度，并再次运行`PROBE_ACCURACY`命令。理想情况下，该命令将报告相同的结果。与上述相同，如果探针确实存在温度偏差，则要小心始终以恒定的温度使用探针。
