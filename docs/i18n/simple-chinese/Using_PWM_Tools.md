# 使用 PWM 工具

本文档描述了如何使用 `pwm_tool` 和一些宏来设置 PWM 控制的激光或主轴。

## 工作原理

通过重新利用打印头风扇的 PWM 输出，您可以控制激光或主轴。
如果您使用可切换的打印头，例如 E3D 工具更换器或 DIY 解决方案，这将非常有用。
通常，像 LaserWeb 这样的 CAM 工具可以配置为使用 `M3-M5` 命令，
它们分别代表 _主轴速度 CW_ (`M3 S[0-255]`)、
_主轴速度 CCW_ (`M4 S[0-255]`) 和 _主轴停止_ (`M5`)。

**警告：** 当驱动激光时，请保持所有能想到的安全预防措施！二极管激光通常是反相的。
这意味着，当 MCU（微控制器）重启时，在 MCU 重新启动所需的时间内，激光将处于
_完全开启_ 状态。为了安全起见，建议在激光通电时 _始终_ 佩戴合适的正确波长激光护目镜；
并在不需要时断开激光。此外，您应配置安全超时，以便当您的主机或 MCU 遇到错误时，工具将停止。

有关示例配置，请参见 [config/sample-pwm-tool.cfg](../config/sample-pwm-tool.cfg)。

## 命令

`M3/M4 S<value>` ：设置 PWM 占空比。值在 0 到 255 之间。
`M5` ：将 PWM 输出停止到关闭值。

## Laserweb 配置

如果您使用 Laserweb，一个可用的配置是：

    GCODE START:
        M5            ; 禁用激光
        G21           ; 将单位设置为 mm
        G90           ; 绝对定位
        G0 Z0 F7000   ; 设置非切割速度

    GCODE END:
        M5            ; 禁用激光
        G91           ; 相对定位
        G0 Z+20 F4000 ;
        G90           ; 绝对定位

    GCODE HOMING:
        M5            ; 禁用激光
        G28           ; 所有轴归位

    TOOL ON:
        M3 $INTENSITY

    TOOL OFF:
        M5            ; 禁用激光

    LASER INTENSITY:
        S
