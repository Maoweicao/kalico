# 赞助商

Kalico是一个由社区驱动的开源项目，可以免费使用（如"免费啤酒"）。虽然我们目前还没有官方赞助商，但我们对社区的慷慨支持深表感谢。每一项贡献都帮助我们继续为所有人改进Kalico。

如果您想支持Kalico，欢迎向我们的Ko-Fi捐款：
👉 [https://ko-fi.com/kalicocrew](https://ko-fi.com/kalicocrew)
