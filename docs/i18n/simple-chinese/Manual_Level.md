# 手动调平

本文档描述用于校准 Z 限位和调整床平整螺钉的工具。

## 校准 Z 限位

准确的 Z 限位位置对于获得高质量打印至关重要。

但请注意，Z 限位开关本身的精度可能是限制因素。如果使用 Trinamic 步进电机驱动器，请考虑启用 [限位器相位](Endstop_Phase.md) 检测以改进开关的精度。

要执行 Z 限位校准，首先归位打印机，然后命令头部移动到至少距床面 5 毫米的 Z 位置（如果尚未这样做），命令头部移动到床中心附近的 XY 位置，然后导航到 OctoPrint 终端选项卡并运行：
```
Z_ENDSTOP_CALIBRATE
```
然后按照 ["纸张测试"](Bed_Level.md#the-paper-test) 中描述的步骤确定喷嘴和床之间在给定位置的实际距离。一旦这些步骤完成，可以 `ACCEPT` 位置并使用以下命令将结果保存到配置文件：
```
SAVE_CONFIG
```

最好在 Z 轴与床相对的末端使用 Z 限位开关。（从床远离归位更强大，因为通常总是可以安全地归位 Z。）但是，如果必须朝向床归位，建议调整限位使其触发在喷嘴仍距床一小段距离（例如 0.5 毫米）时触发。这样做时，应该找到 `Z_ENDSTOP_CALIBRATE` 命令为 Z position_endstop 报告一个小的正值（例如 0.5 毫米）。当限位仍距床一段距离时触发限位会降低无意中发生床碰撞的风险。

某些打印机具有手动调整物理限位开关位置的能力。但是，建议使用 Kalico 在软件中执行 Z 限位定位 - 一旦物理位置在方便的位置，可以通过运行 Z_ENDSTOP_CALIBRATE 或通过手动更新配置文件中的 Z position_endstop 进行任何进一步的调整。

## 调整床平整螺钉

使用床平整螺钉获得良好床平整的秘诀是在床平整过程中利用打印机的高精度运动系统。这通过命令喷嘴到每个床螺钉附近的位置，然后调整该螺钉直到床距喷嘴一定距离来完成。Kalico 有一个工具来协助此操作。为了使用该工具，有必要指定每个螺钉的 XY 位置。

这通过创建 `[bed_screws]` 配置部分完成。例如，它可能看起来类似于：
```
[bed_screws]
screw1: 100, 50
screw2: 100, 150
screw3: 150, 100
```

如果床螺钉在床下，则指定螺钉正上方的 XY 位置。如果螺钉在床外，则指定最接近螺钉但仍在床范围内的 XY 位置。

配置文件准备好后，运行 `RESTART` 加载该配置，然后可以通过运行以下命令启动工具：
```
BED_SCREWS_ADJUST
```

此工具将打印机的喷嘴移动到每个螺钉 XY 位置，然后将喷嘴移动到 Z=0 高度。此时，可以使用"纸张测试"直接调整喷嘴下的床螺钉。查看 ["纸张测试"](Bed_Level.md#the-paper-test) 中描述的信息，但调整床螺钉而不是命令喷嘴到不同的高度。调整床螺钉直到在前后推动纸张时有少量摩擦。

一旦螺钉被调整到感觉到少量摩擦，运行 `ACCEPT` 或 `ADJUSTED` 命令。如果床螺钉需要调整（通常超过大约 1/8 转的螺钉），使用 `ADJUSTED` 命令。如果不需要显著的调整，使用 `ACCEPT` 命令。两个命令都将导致工具继续进行到下一个螺钉。（当使用 `ADJUSTED` 命令时，工具将安排另一个床螺钉调整周期；当所有床螺钉都经过验证不需要任何显著调整时，工具完成成功。）可以使用 `ABORT` 命令提前退出工具。

当打印机具有平面打印表面（例如玻璃）并具有直轨时，此系统工作最佳。成功完成床平整工具后，床应该可以进行打印。

### 细粒度床螺钉调整

如果打印机使用三个床螺钉，所有三个螺钉都在床下，则可能能够执行第二个"高精度"床平整步骤。这是通过命令喷嘴到床随每次床螺钉调整移动更大距离的位置来完成的。

例如，考虑一个床，其螺钉位于位置 A、B 和 C：

![bed_screws](img/bed_screws.svg.png)

对于对螺钉 C 位置的螺钉所做的每个调整，床将沿着由剩余两个床螺钉定义的摆锤摆动（此处显示为绿色线）。在这种情况下，对 C 位置螺钉的每个调整将使床在位置 D 移动比直接在 C 处更大的量。因此，当喷嘴位于位置 D 时，可以进行改进的 C 螺钉调整。

要启用此功能，需要确定其他喷嘴坐标并将其添加到配置文件。例如，它可能看起来像：
```
[bed_screws]
screw1: 100, 50
screw1_fine_adjust: 0, 0
screw2: 100, 150
screw2_fine_adjust: 300, 300
screw3: 150, 100
screw3_fine_adjust: 0, 100
```

启用此功能后，`BED_SCREWS_ADJUST` 工具将首先提示在每个螺钉位置上方进行粗调，一旦这些被接受，它将提示在其他位置进行精细调整。继续在每个位置使用 `ACCEPT` 和 `ADJUSTED`。

## 使用床探针调整床平整螺钉

这是使用床探针校准床平整的另一种方式。要使用它，必须有 Z 探针（BL Touch、电感传感器等）。

要启用此功能，需要确定喷嘴坐标，使得 Z 探针在螺钉上方，然后将其添加到配置文件。例如，它可能看起来像：

```
[screws_tilt_adjust]
screw1: -5, 30
screw1_name: front left screw
screw2: 155, 30
screw2_name: front right screw
screw3: 155, 190
screw3_name: rear right screw
screw4: -5, 190
screw4_name: rear left screw
horizontal_move_z: 10.
speed: 50.
screw_thread: CW-M3
``````

screw1 始终是其他的参考点，因此系统假设 screw1 处于正确的高度。始终首先运行 `G28`，然后运行 `SCREWS_TILT_CALCULATE` - 应该产生类似于以下的输出：
```
Send: G28
Recv: ok
Send: SCREWS_TILT_CALCULATE
Recv: // 01:20 means 1 full turn and 20 minutes, CW=clockwise, CCW=counter-clockwise
Recv: // front left screw (base) : x=-5.0, y=30.0, z=2.48750
Recv: // front right screw : x=155.0, y=30.0, z=2.36000 : adjust CW 01:15
Recv: // rear right screw : y=155.0, y=190.0, z=2.71500 : adjust CCW 00:50
Recv: // read left screw : x=-5.0, y=190.0, z=2.47250 : adjust CW 00:02
Recv: ok
```

这意味着：
- 前左螺钉是参考点，不能改变。
- 前右螺钉必须顺时针转 1 圈和四分之一圈
- 后右螺钉必须逆时针转 50 分钟
- 后左螺钉必须顺时针转 2 分钟（不需要，没关系）

请注意"分钟"是指"时钟面的分钟"。因此，例如，15 分钟是完整转轮的四分之一。

重复该过程几次，直到获得良好的水平床 - 通常当所有调整都在 6 分钟以下时。

如果使用安装在加热头一侧的探针（即，它有 X 或 Y 偏移），请注意调整床倾斜将使之前用倾斜床执行的任何先前探针校准无效。在调整床螺钉后，请务必运行 [探针校准](Probe_Calibrate.md)。

`MAX_DEVIATION` 参数在使用已保存的床网格时很有用，以确保床平整没有从创建网格时漂移太远。例如，`SCREWS_TILT_CALCULATE MAX_DEVIATION=0.01` 可以添加到切片器的自定义启动 gcode 中加载网格之前。如果超过配置的限制（此示例中为 0.01mm），它将中止打印，给用户调整螺钉和重新启动打印的机会。

`DIRECTION` 参数在只能单向转动床调整螺钉时很有用。例如，您可能有从最低（或最高）可能位置开始绷紧的螺钉，只能单向转动以提高（或降低）床。如果只能顺时针转动螺钉，运行 `SCREWS_TILT_CALCULATE DIRECTION=CW`。如果只能逆时针转动，运行 `SCREWS_TILT_CALCULATE DIRECTION=CCW`。将选择合适的参考点，以便可以通过以给定方向转动所有螺钉来调平床。
