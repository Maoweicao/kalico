# 床面平整化

床面平整化（有时也称为"床面调平"）对于获得高质量打印至关重要。如果床面没有正确"平整"，会导致床面粘附不良、"翘曲"以及整个打印过程中的微妙问题。本文档是使用 Kalico 进行床面平整化的指南。

理解床面平整化的目标很重要。如果打印机在打印过程中被命令移动到位置 `X0 Y0 Z10`，那么目标是打印机的喷嘴与打印机床面之间的距离正好是 10mm。进一步，如果打印机随后被命令移动到位置 `X50 Z10`，那么目标是喷嘴在整个水平移动过程中与床面保持准确的 10mm 距离。

为了获得良好的打印质量，打印机应该被校准，使得 Z 方向距离的精度在大约 25 微米（0.025mm）以内。这是一个很小的距离——明显小于典型人类头发的宽度。这个尺度无法通过"目视"来测量。微妙的影响（如热膨胀）会影响这个尺度上的测量。获得高精度的秘诀是使用可重复的过程，并使用一种利用打印机自身运动系统高精度的平整化方法。

## 选择合适的校准机制

不同类型的打印机使用不同的床面平整化方法。所有这些最终都依赖于"纸张测试"（如下所述）。但是，特定类型打印机的实际过程在其他文档中描述。

在运行任何这些校准工具之前，请务必运行[配置检查文档](Config_checks.md)中描述的检查。在执行床面平整化之前，有必要验证基本的打印机运动。

对于配备"自动 Z 探针"的打印机，请务必按照[探针校准](Probe_Calibrate.md)文档中的说明校准探针。对于 Delta 打印机，请参阅[Delta 校准](Delta_Calibrate.md)文档。对于配备床面螺钉和传统 Z 限位开关的打印机，请参阅[手动调平](Manual_Level.md)文档。

在校准期间，可能需要将打印机的 Z `position_min` 设置为负数（例如，`position_min = -2`）。打印机即使在校准程序期间也会强制执行边界检查。设置负数允许打印机移动到低于床面标称位置的位置，这可能有助于确定实际床面位置。

## "纸张测试"

主要的床面校准机制是"纸张测试"。它涉及在打印机的床面和喷嘴之间放置一张普通的"复印纸"，然后将喷嘴命令移动到不同的 Z 高度，直到在来回推动纸张时感觉到轻微的摩擦力。

即使拥有"自动 Z 探针"，理解"纸张测试"也很重要。探针本身通常需要进行校准才能获得良好的结果。该探针校准是使用"纸张测试"完成的。

为了执行纸张测试，用一对剪刀剪下一张小的矩形纸张（例如 5x3 厘米）。纸张通常厚度约为 100 微米（0.100mm）。（纸张的确切厚度并不重要。）

纸张测试的第一步是检查打印机的喷嘴和床面。确保喷嘴或床面上没有塑料（或其他碎屑）。

**检查喷嘴和床面以确保没有塑料！**

如果总是在特定的胶带或打印表面上打印，那么可以在该胶带/表面就位的情况下执行纸张测试。但是，请注意胶带本身有厚度，不同的胶带（或任何其他打印表面）将影响 Z 测量。请务必重新运行纸张测试，以测量使用中的每种表面类型。

如果喷嘴上有塑料，则加热挤出机并使用金属镊子将其移除。等待挤出机完全冷却到室温后再继续纸张测试。当喷嘴冷却时，使用金属镊子移除可能渗出的任何塑料。

**始终在喷嘴和床面都处于室温时执行纸张测试！**

当喷嘴受热时，其位置（相对于床面）由于热膨胀而改变。这种热膨胀通常约为 100 微米，这大约与典型打印纸张的厚度相同。确切的热膨胀量并不重要，就像纸张的确切厚度并不重要一样。首先假设两者相等（请参阅下面关于确定两个距离之间差异的方法）。

当喷嘴受热时进行校准似乎很奇怪，而目标是在受热时保持一致的距离。但是，如果在喷嘴受热时进行校准，往往会在纸张上留下少量熔融塑料，这会改变所感觉的摩擦量。这使得获得良好的校准更加困难。当床面/喷嘴很热时进行校准也大大增加了烧伤自己的风险。热膨胀量是稳定的，所以在后续校准过程中很容易解释。这个。

**使用自动化工具来确定精确的 Z 高度！**

Kalico 提供了几个辅助脚本（例如 MANUAL_PROBE、Z_ENDSTOP_CALIBRATE、PROBE_CALIBRATE、DELTA_CALIBRATE）。请参阅[上述文档](#choose-the-appropriate-calibration-mechanism)选择其中之一。

在 OctoPrint 终端窗口中运行相应的命令。该脚本将在 OctoPrint 终端输出中提示用户交互。它看起来像这样：
```
Recv: // Starting manual Z probe. Use TESTZ to adjust position.
Recv: // Finish with ACCEPT or ABORT command.
Recv: // Z position: ?????? --> 5.000 <-- ??????
```

喷嘴的当前高度（如打印机目前所理解的）显示在"--> <--"之间。右边的数字是最后一次探针尝试的高度，大于当前高度，左边是最后一次探针尝试的高度小于当前高度（或如果没有进行过尝试则为"??????"）。

将纸张放在喷嘴和床面之间。折叠纸张的一个角可能会很有用，这样更容易抓住。（尽量不要在来回移动纸张时按下床面。）

![paper-test](img/paper-test.jpg)

使用 TESTZ 命令请求喷嘴向纸张靠近。例如：
```
TESTZ Z=-.1
```

TESTZ 命令将从喷嘴的当前位置移动相对距离。（因此，`Z=-.1` 请求喷嘴向床面靠近 0.1mm。）喷嘴停止移动后，来回推动纸张以检查喷嘴是否与纸张接触并感觉摩擦量。继续发出 TESTZ 命令，直到感觉到用纸张测试时有轻微的摩擦力。

如果发现摩擦力过大，则可以使用正 Z 值将喷嘴向上移动。还可以使用 `TESTZ Z=+` 或 `TESTZ Z=-` 来"二分"最后位置——即移动到两个位置之间的中点。例如，如果从 TESTZ 命令中收到以下提示：
```
Recv: // Z position: 0.130 --> 0.230 <-- 0.280
```
然后 `TESTZ Z=-` 会将喷嘴移动到 Z 位置 0.180（0.130 和 0.230 之间的中点）。可以使用此功能帮助快速缩小到一致的摩擦力。还可以使用 `Z=++` 和 `Z=--` 直接返回到过去的测量——例如，在上述提示之后，`TESTZ Z=--` 命令会将喷嘴移动到 Z 位置 0.130。

找到少量摩擦力后，运行 ACCEPT 命令：
```
ACCEPT
```
这将接受给定的 Z 高度并继续使用给定的校准工具。

感觉到的确切摩擦量并不重要，就像热膨胀量和纸张的确切宽度并不重要一样。只需尽量在每次运行测试时获得相同的摩擦量。

如果在测试过程中出现问题，可以使用 `ABORT` 命令退出校准工具。

## 确定热膨胀

成功执行床面平整化后，可以继续计算"热膨胀"、"纸张厚度"和"纸张测试期间感觉的摩擦力"的更精确的组合影响值。

这种类型的计算通常不需要，因为大多数用户发现简单的"纸张测试"提供了良好的结果。

进行这种计算的最简单方法是打印一个在所有边上都有直墙的测试对象。[docs/prints/square.stl](prints/square.stl) 中的大型空心正方形可以用于此。切片对象时，确保切片器对第一层使用与所有后续层相同的层高和挤压宽度。使用粗糙的层高（层高应该约为喷嘴直径的 75%）并且不使用边框或筏子。

打印测试对象，等待其冷却，然后从床面上取下。检查对象的最底层。（用手指或指甲沿底边可能也很有用。）如果发现底层沿着对象的所有边都略微突出，则表明喷嘴比应该更靠近床面。可以发出 `SET_GCODE_OFFSET Z=+.010` 命令来增加高度。在后续打印中，可以检查这种行为并根据需要进行进一步调整。这种类型的调整通常为 10 微米（0.010mm）。

如果底层始终看起来比后续层窄，则可以使用 SET_GCODE_OFFSET 命令进行负 Z 调整。如果不确定，则可以减少 Z 调整，直到打印的底层出现小的隆起，然后后退直到其消失。

应用所需 Z 调整的最简单方法是创建 START_PRINT g 代码宏，安排切片器在每次打印开始时调用该宏，然后将 SET_GCODE_OFFSET 命令添加到该宏。有关详细信息，请参阅[切片器](Slicers.md)文档。
