# 共振补偿

Kalico支持输入整形 - 一种可用于减少打印中的振铃（也称为回声、鬼影或波纹）的技术。振铃是一种表面打印缺陷，通常情况下，边缘等元素在打印表面上重复出现，呈现出微妙的"回声"：

|![振铃测试](img/ringing-test.jpg)|![3D Benchy](img/ringing-3dbenchy.jpg)|

振铃是由于打印方向快速变化导致的打印机中的机械振动引起的。请注意，振铃通常具有机械起源：打印机框架刚度不足、皮带松弛或弹性过大、机械部件对齐问题、移动质量过大等。如果可能，应首先检查和修复这些问题。

[输入整形](https://en.wikipedia.org/wiki/Input_shaping)是一种开环控制技术，它创建一个消除自身振动的指挥信号。输入整形需要进行一些调整和测量，才能启用。除了振铃外，输入整形通常会减少打印机的整体振动和颤动，并可能改善Trinamic步进电机的stealthChop模式的可靠性。

## 调整

基本调整需要通过打印测试模型来测量打印机的振铃频率。

将振铃测试模型（位于[docs/prints/ringing_tower.stl](prints/ringing_tower.stl)）在切片机中进行切片：

* 建议的层高为0.2或0.25毫米。
* 填充和顶层可以设置为0。
* 使用1-2个周边，或更好的是光滑的花瓶模式，底部1-2毫米。
* 对于**外部**周边，使用足够高的速度，约80-100毫米/秒。
* 确保最小层时间**最多**为3秒。
* 确保在切片机中禁用任何"动态加速度控制"。
* 不要旋转模型。模型背面有X和Y标记。
 请注意标记与打印机轴相比的不寻常位置 - 这不是错误。标记可以在调整过程中稍后用作参考，因为它们显示测量对应的轴。

### 振铃频率

首先，测量**振铃频率**。

1. 如果`square_corner_velocity`参数已更改，请将其恢复为5.0。不建议在使用输入整形器时增加它，因为它可能在零件中造成更多平滑 - 最好使用更高的加速度值。
2. 通过发出以下命令禁用`minimum_cruise_ratio`功能：`SET_VELOCITY_LIMIT MINIMUM_CRUISE_RATIO=0`
3. 禁用压力提前：`SET_PRESSURE_ADVANCE ADVANCE=0`
4. 如果已将`[input_shaper]`部分添加到printer.cfg，请执行`SET_INPUT_SHAPER SHAPER_FREQ_X=0 SHAPER_FREQ_Y=0`命令。如果收到"Unknown command"错误，可以安全地在此时忽略它并继续测量。
5. 执行命令：
   `TUNING_TOWER COMMAND=SET_VELOCITY_LIMIT PARAMETER=ACCEL START=1500 STEP_DELTA=500 STEP_HEIGHT=5`
   基本上，我们通过设置不同的大加速度值来尝试使振铃更明显。此命令将从1500毫米/秒²开始每5毫米增加加速度：1500毫米/秒²、2000毫米/秒²、2500毫米/秒²，以此类推，直到最后一个频段的7000毫米/秒²。
6. 用建议的参数打印测试模型。
7. 如果振铃清晰可见，并且您看到打印机的加速度过高（例如打印机摇晃过多或开始跳步），可以提前停止打印。
8. 使用模型背面的X和Y标记作为参考。来自X标记一侧的测量应用于X轴*配置*，Y标记 - 用于Y轴配置。测量X标记部分上靠近凹口的多个振荡之间的距离*D*（毫米），最好跳过第一个或两个振荡。为了更容易测量振荡之间的距离，请先标记振荡，然后用直尺或卡尺测量标记之间的距离：

    |![标记振铃](img/ringing-mark.jpg)|![测量振铃](img/ringing-measure.jpg)|

9. 计算测量的距离*D*对应多少个振荡*N*。如果不确定如何计算振荡，请参考上面的图片，该图片显示*N* = 6个振荡。
10. 计算X轴的振铃频率为*V* · *N* / *D*（赫兹），其中*V*是外周边的速度（毫米/秒）。对于上面的示例，我们标记了6个振荡，测试以100毫米/秒的速度打印，所以频率是100 * 6 / 12.14 ≈ 49.4赫兹。
11. 对Y标记也做(8) - (10)。

请注意，测试打印上的振铃应遵循弯曲凹口的图案，如上面的图片所示。如果不是，则此缺陷不是真正的振铃，而是来自不同起源 - 要么机械，要么挤出机问题。应在启用和调整输入整形器之前首先修复它。

如果由于振荡之间的距离不稳定而导致测量不可靠，则可能意味着打印机在同一轴上有多个共振频率。可以尝试按照[不可靠的振铃频率测量](#不可靠的振铃频率测量)部分中描述的调整过程，仍然可以从输入整形技术中获得一些东西。

振铃频率可能取决于模型在打印床内的位置和Z高度，*特别是在三角洲打印机上*；可以检查在测试模型侧面的不同位置和不同高度处是否看到频率的差异。在这种情况下，可以计算X和Y轴上的平均振铃频率。

如果测量的振铃频率非常低（约20-25赫兹以下），在继续进行进一步的输入整形调整之前，投资于加强打印机或减少移动质量可能是个好主意，然后重新测量频率。对于许多流行的打印机型号，通常已经有一些解决方案可用。

请注意，如果对打印机进行影响移动质量或改变系统刚度的更改，振铃频率可能会改变，例如：

* 工具头上安装、卸载或更换了工具，改变了其质量，例如为直接挤出机安装了新的（更重或更轻的）步进电机，或安装了新的热端、添加了重型风扇和管道等。
* 皮带拧紧。
* 安装了一些增加框架刚度的附加组件。
* 在床式滑块打印机上安装了不同的床，或添加了玻璃等。

如果进行了这些更改，至少测量振铃频率以查看它们是否已更改是个好主意。

### 输入整形器配置

在测量了X和Y轴的振铃频率后，可以将以下部分添加到`printer.cfg`：
```
[input_shaper]
shaper_freq_x: ...  # 测试模型X标记的频率
shaper_freq_y: ...  # 测试模型Y标记的频率
```

对于上面的示例，我们得到shaper_freq_x/y = 49.4。

### 选择输入整形器

Kalico支持多种输入整形器。它们在确定共振频率时对错误的敏感性和它们在打印零件中引起多少平滑方面有所不同。此外，某些整形器（如2HUMP_EI和3HUMP_EI）通常不应与shaper_freq =共振频率一起使用 - 它们从不同的考虑因素配置以一次减少多个共振。

对于大多数打印机，可以推荐MZV或EI整形器。本部分描述了在两者之间进行选择的测试过程，并找出其他一些相关参数。

按如下方式打印振铃测试模型：

1. 重启固件：`RESTART`
2. 准备测试：`SET_VELOCITY_LIMIT MINIMUM_CRUISE_RATIO=0`
3. 禁用压力提前：`SET_PRESSURE_ADVANCE ADVANCE=0`
4. 执行：`SET_INPUT_SHAPER SHAPER_TYPE=MZV`
5. 执行命令：
   `TUNING_TOWER COMMAND=SET_VELOCITY_LIMIT PARAMETER=ACCEL START=1500 STEP_DELTA=500 STEP_HEIGHT=5`
6. 用建议的参数打印测试模型。

如果此时没有看到振铃，则可以推荐使用MZV整形器。

如果确实看到一些振铃，使用[振铃频率](#振铃频率)部分中描述的步骤(8)-(10)重新测量频率。如果频率与之前获得的值显著不同，则需要更复杂的输入整形器配置。可以参考[输入整形器](#输入整形器)部分的技术细节。否则，继续下一步。

现在尝试EI输入整形器。要尝试它，重复上面的步骤(1)-(6)，但在第4步执行以下命令：
`SET_INPUT_SHAPER SHAPER_TYPE=EI`。

比较两个使用MZV和EI输入整形器的打印。如果EI显示比MZV明显更好的结果，使用EI整形器，否则更喜欢MZV。请注意，EI整形器将在打印的零件中造成更多平滑（有关详细信息，请参阅下一部分）。将`shaper_type: mzv`（或ei）参数添加到[input_shaper]部分，例如：
```
[input_shaper]
shaper_freq_x: ...
shaper_freq_y: ...
shaper_type: mzv
```

关于整形器选择的几点注意：

* EI整形器可能更适合床式滑块打印机（如果共振频率和产生的平滑允许）：随着更多灯丝沉积在移动的床上，床的质量增加，共振频率会降低。由于EI整形器对共振频率变化更稳健，它在打印大部分时可能效果更好。
* 由于三角洲运动学的性质，共振频率在构建体积的不同部分可能差异很大。因此，EI整形器可能比MZV或ZV更适合三角洲打印机，应该考虑使用。如果共振频率足够大（超过50-60赫兹），甚至可以尝试测试2HUMP_EI整形器（通过以`SET_INPUT_SHAPER SHAPER_TYPE=2HUMP_EI`运行建议的测试），但在启用它之前检查[下面部分](#选择max_accel)中的考虑。

### 选择max_accel

从上一步中选择的整形器应该已经有打印的测试（如果没有，用[建议的参数](#调整)打印测试模型，禁用压力提前`SET_PRESSURE_ADVANCE ADVANCE=0`，并启用调整塔`TUNING_TOWER COMMAND=SET_VELOCITY_LIMIT PARAMETER=ACCEL START=1500 STEP_DELTA=500 STEP_HEIGHT=5`）。请注意，在非常高的加速度下，根据共振频率和所选的输入整形器（例如EI整形器比MZV产生更多平滑），输入整形可能会产生过多平滑和零件圆角。因此，应选择max_accel以防止这种情况。另一个可能影响平滑的参数是`square_corner_velocity`，因此不建议将其增加到默认的5毫米/秒以上以防止增加平滑。

为了选择合适的max_accel值，检查所选输入整形器的模型。首先，记下在哪个加速度振铃仍然很小 - 您对其感到满意。

接下来，检查平滑。为帮助解决这个问题，测试模型在墙上有一个小间隙（0.15毫米）：

![测试间隙](img/smoothing-test.png)

随着加速度的增加，平滑也会增加，打印中的实际间隙也会增加：

![整形器平滑](img/shaper-smoothing.jpg)

在这张图片中，加速度从左到右增加，从3500毫米/秒²（从左边数第5个频段）开始，间隙开始增加。因此在这种情况下max_accel = 3000（毫米/秒²）的好值是避免过度平滑。

在测试打印中注意间隙仍然很小时的加速度。如果看到凸起，但在高加速度下墙上根本没有间隙，这可能是由于禁用的压力提前，特别是在Bowden挤出机上。在这种情况下，可能需要在启用PA的情况下重复打印。这也可能是结果误校准（太高）的灯丝流，因此检查这一点也是个好主意。

选择两个加速度值（来自振铃和平滑）中的最小值，并将其作为`max_accel`放入printer.cfg。

请注意，可能会发生 - 特别是在低振铃频率下 - EI整形器即使在较低加速度下也会导致过度平滑。在这种情况下，MZV可能是更好的选择，因为它可能允许更高的加速度值。

在非常低的振铃频率（约25赫兹及以下），即使MZV整形器也可能产生过度平滑。如果是这种情况，可以尝试通过使用`SET_INPUT_SHAPER SHAPER_TYPE=ZV`命令重复[选择输入整形器](#选择输入整形器)部分中的步骤来尝试ZV整形器。ZV整形器应显示比MZV更少的平滑，但对测量振铃频率中的错误更敏感。

另一个考虑是，如果共振频率太低（低于20-25赫兹），可能是个好主意增加打印机刚度或减少移动质量。否则，由于现在平滑过度而不是振铃，加速度和打印速度可能受到限制。

### 微调共振频率

请注意，使用振铃测试模型的共振频率测量的精度对大多数目的都足够了，因此不建议进一步调整。如果仍然想尝试仔细检查结果（例如，如果在启用[input_shaper]后仍然看到一些振铃，使用的频率与之前测量的相同），可以按照本部分中的步骤进行。请注意，如果在启用[input_shaper]后看到不同频率的振铃，本部分将不会有帮助。

假设已用建议的参数切片了振铃模型，为每个轴X和Y完成以下步骤：

1. 准备测试：`SET_VELOCITY_LIMIT MINIMUM_CRUISE_RATIO=0`
2. 确保禁用压力提前：`SET_PRESSURE_ADVANCE ADVANCE=0`
3. 执行：`SET_INPUT_SHAPER SHAPER_TYPE=ZV`
4. 从现有振铃测试模型中选择显示振铃足够好的加速度，并通过以下方式设置：
   `SET_VELOCITY_LIMIT ACCEL=...`
5. 计算`TUNING_TOWER`命令调整`shaper_freq_x`参数所需的参数如下：start = shaper_freq_x * 83 / 132，factor = shaper_freq_x / 66，其中这里的`shaper_freq_x`是`printer.cfg`中的当前值。
6. 执行命令：
   `TUNING_TOWER COMMAND=SET_INPUT_SHAPER PARAMETER=SHAPER_FREQ_X START=start FACTOR=factor BAND=5`
   使用在步骤(5)中计算的`start`和`factor`值。
7. 打印测试模型。
8. 重置原始频率值：
   `SET_INPUT_SHAPER SHAPER_FREQ_X=...`。
9. 找到显示振铃最少的频段，并从下面开始从1计算其数字。
10. 通过旧shaper_freq_x * (39 + 5 * #band-number) / 66计算新的shaper_freq_x值。

以相同的方式为Y轴重复这些步骤，用轴Y替换对X轴的引用（例如，在公式和`TUNING_TOWER`命令中将`shaper_freq_x`替换为`shaper_freq_y`）。

作为示例，假设已为某个轴测量了振铃频率等于45赫兹。这给出start = 45 * 83 / 132 = 28.30和factor = 45 / 66 = 0.6818值用于`TUNING_TOWER`命令。
