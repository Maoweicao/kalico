# 配置检查

本文档提供了一些步骤列表，用于帮助确认 Kalico printer.cfg 文件中的引脚设置。
按照 [安装文档](Installation.md) 中的步骤后，最好按照这些步骤操作。

在本指南中，可能需要对 Kalico 配置文件进行更改。
确保在对配置文件进行每次更改后发出 RESTART 命令，
以确保更改生效（在 Octoprint 终端选项卡中键入"restart"，然后单击"Send"）。
在每次 RESTART 之后发出 STATUS 命令也是个好主意，
以验证配置文件已成功加载。

## 验证温度

首先验证温度是否被正确报告。导航到用户界面中的温度图形部分。
验证喷嘴和床的温度（如适用）是否存在且不增加。
如果增加，请切断打印机的电源。如果温度不准确，
请检查喷嘴和/或床的"sensor_type"和"sensor_pin"设置。

## 验证 M112

导航到命令控制台并在终端框中发出 M112 命令。
此命令请求 Kalico 进入"关闭"状态。它将导致显示错误，
可以在命令控制台中使用 FIRMWARE_RESTART 命令清除错误。
Octoprint 也将需要重新连接。然后导航到温度图形部分，
验证温度继续更新且温度不增加。
如果温度增加，请切断打印机的电源。

## 验证加热器

导航到温度图形部分，在挤出机/工具温度框中输入 50，然后按 Enter。
图形中的挤出机温度应开始增加（大约 30 秒左右）。
然后进入挤出机温度下拉框并选择"Off"。几分钟后，
温度应开始回到其初始室温值。如果温度不增加，
则验证配置中的"heater_pin"设置。

如果打印机有加热床，则再次使用床执行上述测试。

## 验证步进电机使能引脚

验证打印机的所有轴都可以手动自由移动（步进电机被禁用）。
如果不能，发出 M84 命令禁用电机。如果任何轴仍然不能自由移动，
则验证给定轴的步进"enable_pin"配置。在大多数普通步进电机驱动器上，
电机使能引脚是"低电平有效"的，因此使能引脚应在引脚前有"!"
（例如，"enable_pin: !PA1"）。

## 验证限位开关

手动移动所有打印机轴，使它们都不与限位开关接触。
通过命令控制台发送 QUERY_ENDSTOPS 命令。它应响应所有配置的限位开关的当前状态，
并且它们都应报告"open"状态。对于每个限位开关，在手动触发限位开关时
重新运行 QUERY_ENDSTOPS 命令。QUERY_ENDSTOPS 命令应将限位开关报告为"TRIGGERED"。

如果限位开关显示为反向（触发时报告"open"，反之亦然），
则在引脚定义中添加"!"（例如，"endstop_pin: ^!PA2"），
或如果已经存在"!"则移除它。

如果限位开关根本没有改变，则通常表示限位开关连接到不同的引脚。
但是，它也可能需要更改引脚的上拉设置（限位开关 pin 名称开头的"^"
——大多数打印机将使用上拉电阻，"^"应该存在）。

## 验证步进电机

使用 STEPPER_BUZZ 命令验证每个步进电机的连接性。
首先手动将给定轴定位到中点，然后在命令控制台中运行 `STEPPER_BUZZ STEPPER=stepper_x`。
STEPPER_BUZZ 命令将导致给定的步进器向正方向移动一毫米，
然后返回到其起始位置。（如果限位开关定义在 position_endstop=0，
则在每个运动的开始，步进器将远离限位开关。）
它将执行此振荡十次。

如果步进器根本不移动，则验证步进器的"enable_pin"和"step_pin"设置。
如果步进器电机移动但不返回到其原始位置，则验证"dir_pin"设置。
如果步进器电机以不正确的方向振荡，则通常表示轴的"dir_pin"需要反向。
这是通过在 printer config 文件中的"dir_pin"中添加"!"（或如果已经存在则移除）来完成的。
如果电机移动明显超过或明显少于一毫米，则验证"rotation_distance"设置。

对配置文件中定义的每个步进电机运行上述测试。
（将 STEPPER_BUZZ 命令的 STEPPER 参数设置为要测试的配置部分的名称。）
如果挤出机中没有耗材，可以使用 STEPPER_BUZZ 验证挤出机电机连接性（使用 STEPPER=extruder）。
否则，最好单独测试挤出机电机（请参阅下一部分）。

验证所有限位开关和验证所有步进电机后，应测试归位机制。
发出 G28 命令对所有轴进行归位。如果不能正确归位，请切断打印机的电源。
如有必要，重新运行限位开关和步进电机验证步骤。

## 验证挤出机电机

要测试挤出机电机，需要将挤出机加热到打印温度。导航到温度图形部分，
从温度下拉框中选择目标温度（或手动输入适当的温度）。
等待打印机达到所需温度。然后导航到命令控制台，
单击"Extrude"按钮。验证挤出机电机朝正确方向转动。
如果没有，请参阅前一部分中的故障排查提示以确认挤出机的"enable_pin"、
"step_pin"和"dir_pin"设置。

## 校准 PID 设置

Kalico 支持对挤出机和床加热器的 [PID 控制](https://en.wikipedia.org/wiki/PID_controller)。
为了使用此控制机制，需要在每台打印机上校准 PID 设置
（在其他固件或示例配置文件中找到的 PID 设置通常效果不佳）。

要校准挤出机，导航到命令控制台并运行 PID_CALIBRATE 命令。
例如：`PID_CALIBRATE HEATER=extruder TARGET=170`

在调谐测试完成时运行 `SAVE_CONFIG` 以使用新的 PID 设置更新 printer.cfg 文件。

如果打印机有加热床且支持由 PWM（脉宽调制）驱动，
则建议对床使用 PID 控制。（当床加热器使用 PID 算法控制时，
它每秒可能打开和关闭十次，这对于使用机械开关的加热器可能不合适。）
典型的床 PID 校准命令是：`PID_CALIBRATE HEATER=heater_bed TARGET=60`
